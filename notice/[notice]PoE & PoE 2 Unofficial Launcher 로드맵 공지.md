## 1.x.0 Loadmap (기타 우선순위)

### 기능 개선

- 디스코드 관련
  - [ ] Q&A 채널 운영
    - [ ] 바이러스 오탐지 관련 (윈도우 디펜더 업데이트 가이드, 백신 예외 등록 가이드 등)

  - [ ] 게임내에서 동작하는 서드파티 프로세스를 관리

  - [ ] Ultrawide 해상도 패치 ( own risk 경고 문구 포함 및 해당 해상도 사용 시에만 추천 )

  - [ ] 거래소 오버레이 ( 인게임 거래소가 불편함. 홈페이지를 그냥 띄우기 )
