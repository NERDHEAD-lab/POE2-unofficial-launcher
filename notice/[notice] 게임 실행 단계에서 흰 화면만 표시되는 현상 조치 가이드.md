# [공지] 게임 실행 후 홈 화면 노출 및 지연 경고 현상 수정 안내

안녕하세요. 개발자 NERDHEAD 입니다.

v0.9.8 버전에서 게임 실행 절차 중 지연 발생 시 무반응이던 문제(화이트 스크린 등)를 해결하기 위해 도입된 기능(지연 발생 시 숨겨진 창을 자동으로 표시)이, 특정 상황에서 정상적인 실행 과정 중에도 약 30초가 지나면 창이 표시되어 버리는 문제가 있었습니다.

**v0.9.10 버전에서 해당 문제를 수정하였습니다.**

### 주요 변경 사항

1. **타임아웃 로직 수정**: 정상적인 실행 과정에서 창이 불필요하게 표시되는 현상을 수정했습니다.
2. **타임아웃 시간 최적화**: 대기 시간을 30초에서 10초로 단축하여 지연 발생 시 더욱 빠르게 대응할 수 있도록 개선했습니다.
3. **지연 정보 강화**: 타임아웃 발생 시 표시되는 경고창에 **현재 문제가 발생한 URL**을 함께 표시하여 원인 파악이 쉽도록 개선했습니다.

런처를 완전히 종료 후 재시작하시면 최신 버전으로 업데이트를 받으실 수 있습니다.

쾌적한 환경을 제공할 수 있도록 더욱 노력하겠습니다.

감사합니다.
