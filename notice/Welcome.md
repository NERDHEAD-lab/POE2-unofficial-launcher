자세한 내용은 중요 공지사항의 `PoE & PoE 2 Unofficial Launcher 로드맵 공지`를 참고해 주시기 바랍니다.

## 개발 배경

해당 비공식 런쳐는 제가 PoE2를 플레이하며 불편한 점을 조치하기 위해 개발하던 몇 가지 기능들을 모아서 개발하였습니다.
[POE2 카카오 업데이트 오류 자동 수정 도구](https://github.com/NERDHEAD-lab/POE2-KG-Client-Patch-Butler)

```
- 게임 패치 마다 반복해서 발생하는 `Transferred a partial file` 오류를 자동으로 수정
  - 오류 자동 수정 후, 게임을 자동으로 실행 할 수 있도록 지원
- 게임 실행 간 DaumGameStarter에서 발생하는 사용자 계정 컨트롤 팝업을 자동으로 닫기
```

[POE & POE2 빠른 실행 (Kakao)](https://github.com/NERDHEAD-lab/POE2-quick-launch-for-kakao)

```
- 홈페이지를 통해 게임 실행 시 다양하게 마주하는 팝업들을 자동으로 진행
- 개인적으로 생각했을 때 게이머가 가장 필요로 하는 패치노트를 홈페이지에서 바로 볼 수 없어 아쉬운 점을 개선
```

다들 불편한 경험을 PoE2 출시 이후 겪고 있을 것이라고 생각했고,
제 개인의 불편함을 해결하기 위해 개발하던 기능들을 모아 런쳐를 개발하게 되었습니다.

안타깝게도 PoE2가 인생게임이 된 저로써는,
많은 유저들이 게임 실행 부터 진입장벽으로 느끼지 않고 적어도 게임은 즐길 수 있도록 돕고 싶었습니다.
결국 게임은 사람이 많아야 재미있으니까요.

모두의 편안한 유배자 생활을 위해 이타적으로 개발했다기 보다
이 게임 말도 많고 탈도 많지만, 제발 나랑 게임 같이해줘-의 마음으로 개발한거라 보시면 됩니다.

물론, 덕분에 이번 시즌은 사원 완공도 못하고 종일 코딩만 하고 있어서 본말이 전도된 느낌이긴 합니다만...ㅋㅋ

## 후원 관련

런쳐 이전/이후로 다양하게 후원 주신분들 정말 감사합니다.

기존 빠른실행 플러그인이나, 오류 자동 수정 도구 등으로 받은 후원의 경우 제 일용할 카페인이 되어 혈관속을 잘 돌아다니고 있습니다.

런쳐를 통해 모인 후원 금액은 런쳐의 코드 사인 발급에 우선적으로 사용 할 예정입니다.
코드 사인 발급에 필요한 금액은 1,120,000원 정도로 3년의 기간동안 유효합니다.

`2026-02-13 기준`
| 모금 금액 | 목표 금액 |
| ---------- | ---------- |
| 237,000원 | 1,120,000원 |

대충 백신이 오탐지 하는 이유가 출처를 알 수 없는 프로그램인 이유가 크고,
해결하려면 코드 사인이라는 인증서를 발급 받아야하는데 개인이 발급 받기엔 비용이 비싼 문제가 좀 있긴 합니다.

따라서 적극적으로 후원을 모금하고 있으나 1.0.0 릴리즈 전까지 후원 금액이 부족하다면,
그냥 취직해서 사비 털어서 코드 사인을 발급 받지 않을까 싶습니다.
