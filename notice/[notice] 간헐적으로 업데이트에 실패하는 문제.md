# 간헐적으로 런쳐 업데이트에 실패하는 문제 (해결)

안녕하세요. 개발자 NERDHEAD 입니다.

일부 사용자로부터 런쳐 업데이트 후 재시작을 클릭하면
종료 후 시작되지 않으며, 업데이트 또한 진행 되지 않는 현상을 보고 받았습니다.

우선 해당 이슈가 있는 경우 아래의 Github Release 링크를 통해 최신 setup으로 수동 설치 부탁드립니다.

### [최신 런쳐 다운로드](https://github.com/NERDHEAD-LAB/POE2-Unofficial-Launcher/releases)

해당 이슈는 1.0.2 이후 버전에서는 발생 하지 않을 예정입니다.

현재 설치 된 버전이 1.0.2 미만 일 경우 해당 문제 발생 시 Github에서 setup을 수동으로 다운받아 설치 부탁드립니다.

죄송합니다.
