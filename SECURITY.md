# Security Policy

## Supported Versions

Currently, we support the latest release version of the launcher.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it via the GitHub Issues tab or contact the maintainers directly.

We appreciate your efforts to responsibly disclose your findings, and will make every effort to acknowledge your contributions.
